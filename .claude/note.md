# 주석 작성 가이드

이 문서는 Bigtablet Design System의 주석 작성 기준을 정의합니다.

## 기본 원칙
- 모든 함수는 JSDoc을 작성한다.
- 주석은 한국어로 작성한다.
- 코드의 흐름과 의도를 설명한다.
- 표준 JSDoc 태그만 사용한다.

## 적용 범위
- 컴포넌트, 훅, 헬퍼 함수 등 명시적으로 선언된 함수
- 제어/비제어 로직, 이벤트 핸들러, 상태 전환 등 동작 흐름이 있는 함수
- 인라인 콜백(예: JSX 내 익명 함수)은 필요할 때만 간단 주석으로 보완한다

## JSDoc 형식
- 요약 1~2문장으로 목적/흐름 설명
- 매개변수는 `@param`으로 설명
- 반환값은 `@returns`로 설명
- 사용 중단은 `@deprecated`로 명시

## 예시
```ts
/**
 * 셀렉트 값을 갱신하고 변경 이벤트를 전파한다.
 * @param next 다음 값
 * @returns void
 */
const setValue = (next: string | null) => {
  // ...
};
```
